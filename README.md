# strings-js


This is a library that allows users to convert strings.


## String Prototype Extension:

1. ```.capitalize()```
1. ```.allCaps()```
1. ```.capitalizeWords()```
1. ```.removeExtraSpaces()```
1. ```.kabobCase()```
1. ```.snakeCase()```
1. ```.camelCase()```
